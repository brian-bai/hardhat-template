# Hardhat Project Template


```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat coverage
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```

## User defined tasks in scripts/tasks.js:
```
  accounts      Prints the list of accounts
  address       Print an address from a private key
  balance       Prints an account balance
  blockNumber   Prints the current block number
  checkenv      Prints the env
  provider      Prints the provider infomaiotn
  receipt       Get Transaction Receipt
  tokenEvent    Print user defined token transfer events in current block
  tokens        Prints balance of user defined ERC20 Token
```
